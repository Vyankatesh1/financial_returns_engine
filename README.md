# Financial Returns Engine

## Overview
This project transforms, validates, and calculates financial returns using monthly portfolio data.
It is designed to be simple, readable, and production-ready.

## Metrics Supported
- Total Return
- CAGR
- Annualized Return
- Volatility
- Sharpe Ratio
- NPV
- IRR

## Design Decisions
- Monthly-based calculations for better accuracy
- Explicit validation to prevent financial errors
- Simple logic that is easy to explain

## Complexity
- Time Complexity: O(n)
- Space Complexity: O(n)

## How to Run
```bash
pip install -r requirements.txt
python examples/sample_run.py
```
